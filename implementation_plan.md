# План реализации Telegram Модератора

Этот проект представляет собой Telegram-бота для модерации каналов и чатов.

## Основные функции:
1.  **Модерация сообщений**:
    *   Удаление ссылок (настраиваемо).
    *   Фильтрация нецензурной лексики (черный список слов).
    *   Анти-спам и анти-флуд.
2.  **Управление пользователями**:
    *   Команды: `/ban`, `/kick`, `/mute`, `/unmute`.
    *   Система предупреждений (`/warn`).
3.  **Автоматизация**:
    *   Приветствие новых участников.
    *   Удаление системных сообщений (о входе/выходе).
4.  **Поддержка каналов**:
    *   Удаление сообщений от имени других каналов.

## Технологический стек:
*   **Язык**: Python 3.10+
*   **Библиотека**: `aiogram` (v3)
*   **Хранилище параметров**: JSON или SQLite (для начала - JSON).

## Структура проекта:
*   `main.py` — точка входа.
*   `config.py` — конфигурация.
*   `handlers/` — обработчики команд и сообщений.
*   `middlewares/` — фильтры спама.
*   `utils/` — вспомогательные функции.
